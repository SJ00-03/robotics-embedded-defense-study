# Robotics Study Checklist

## Core Robotics Concepts
- [ ] 좌표계(x, y, heading) 및 기본 운동학 이해
- [ ] 휠 오도메트리 오차 누적 원인 설명 가능
- [ ] PID 제어기(P/I/D 항 역할) 튜닝 기초 이해
- [ ] 센서 융합 기본 개념(신뢰도/노이즈) 설명 가능

## Planning & Control
- [ ] BFS/A* 경로 계획 알고리즘 독해 가능
- [ ] 장애물 회피 FSM 전이 조건 분석 가능
- [ ] 제어 루프 주기와 시스템 안정성 관계 설명 가능

## Software Practice
- [ ] 센서 처리 파이프라인(수집→필터→판단→제어) 설명 가능
- [ ] 로그 기반 디버깅(상태 전이/오류 시점) 수행 가능
- [ ] mock 환경에서 알고리즘 검증 절차 구성 가능
