# Interview Questions (Robotics / Embedded / Safety-Critical)

## C / Embedded
1. `volatile`이 필요한 상황과 불필요한 상황을 구분해서 설명해보세요.
2. ring buffer에서 head/tail 설계 시 full/empty를 어떻게 구분하나요?
3. ISR에서 처리할 일과 main loop에서 처리할 일을 어떻게 나누시겠습니까?
4. static과 extern의 차이와 실제 프로젝트에서의 사용 예를 말해보세요.
5. checksum/CRC는 어떤 오류를 잡고, 어떤 오류는 못 잡을 수 있나요?

## Python / System Software
6. dataclass를 일반 class 대신 사용할 때 장단점은 무엇인가요?
7. queue/deque 중 무엇을 선택할지 기준을 설명해보세요.
8. logging 레벨 정책(DEV/TEST/PROD)을 어떻게 가져가면 좋을까요?

## Robotics / Safety-Critical
9. 오도메트리 기반 위치 추정의 한계와 보완 방법은 무엇인가요?
10. safety-critical 시스템에서 fault detection과 fail-safe 전환 기준을 어떻게 설계하시겠습니까?

## Portfolio Discussion Tips
- 각 답변에 “실제 구현/독해한 코드 사례”를 1개 이상 연결해 설명하기.
- 트레이드오프(성능/복잡도/안전성)를 함께 설명하기.
